# My-Python-Lab
A collection of Python scripts, exercises, and notes from my daily learning process.

# Python Learning Brain Dump

This repository serves as my personal knowledge base for Python. It’s not just a collection of scripts; it's a curated set of modules covering concepts, edge cases, and syntax details that I found challenging or interesting during my journey.

# Why this Repo?
- **Knowledge Storage:** I convert everything I learn (especially the tricky parts) into modular scripts.
- **Quick Reference:** It acts as my personal documentation to look back at when I forget a specific implementation.
- **Continuous Growth:** This is a living project. I update it almost daily as I encounter new Python puzzles.

# How it's Structured
Each file is a standalone module focused on a specific topic. To keep things clean, I follow this simple naming convention:
- `topic_name.py` — Concepts and core logic (e.g., `dict.py`, `set.py`).
- `practice_topic.py` — Deep dives into specific methods or built-in functions.
